# 🎓 Student Performance Analysis

A Python-based data analysis project that uses **Pandas** and **Matplotlib** to analyze student performance data. This project reads student records from a CSV file, performs statistical analysis, calculates performance metrics, and visualizes the results through charts.

---

## 📌 Project Overview

The **Student Performance Analysis** project demonstrates how Python can be used to analyze and visualize student academic performance. It reads data from a CSV file, calculates total and average marks, determines pass/fail status, identifies top performers, and generates insightful visualizations.

---

## ✨ Features

* 📂 Read student data from a CSV file
* 📊 Display dataset information and statistical summary
* ➕ Calculate total marks
* 📈 Calculate average marks
* ✅ Determine Pass/Fail status
* 🏆 Identify the topper
* 🥇 Display Top 3 students
* 📚 Calculate subject-wise average marks
* 📉 Find highest and lowest marks in each subject
* 🔍 Filter high-performing students
* 💾 Save processed data as a new CSV file
* 📊 Generate graphs using Matplotlib

The dataset contains the following fields:

| English    | English Marks             |

## 🚀 How to Run the Project

### 1️⃣ Clone the repository

```bash
git clone https://github.com/SmrutiSurekhaPati-hub/Student-Performance-Analysis.git
```

### 2️⃣ Navigate to the project folder

```bash
cd Student-Performance-Analysis
```

### 3️⃣ Install required libraries

```bash

```

### 4️⃣ Run the project

